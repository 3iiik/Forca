# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 1.x | ✅ |
| < 1.0 | ❌ |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Forca, please do **not** open a public issue.

Instead, send a private report to **security@forca.app** or reach out to the maintainers directly via GitHub Security Advisories.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact or exploit scenario

You will receive a response within **48 hours**. We will keep you informed of the fix timeline and credit you in the release notes if desired.

## Preferred Languages

English, please.
